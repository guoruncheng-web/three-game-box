# 水果消消乐音效文件

## 使用方法

1. 打开 `public/generate_fruit_sounds.html` 在浏览器中
2. 点击"一键生成所有音效"按钮，或单独生成每个音效
3. 下载的音效文件会保存到当前目录

## 音效文件列表

- `fruit-match-click.wav` - 点击音效
- `fruit-match-swap.wav` - 交换音效
- `fruit-match-match.wav` - 匹配音效
- `fruit-match-score.wav` - 得分音效
- `fruit-match-win.wav` - 胜利音效
- `fruit-match-lose.wav` - 失败音效

## 音效说明

### click.wav
- **用途**: 点击水果时的反馈音
- **时长**: 0.1秒
- **频率**: 800Hz 正弦波
- **特点**: 短促清脆

### swap.wav
- **用途**: 滑动交换水果时的音效
- **时长**: 0.15秒
- **频率**: 600Hz → 400Hz 线性下降
- **特点**: 滑动感

### match.wav
- **用途**: 匹配成功时的音效
- **时长**: 0.4秒
- **频率**: C5-E5-G5 和弦 (523.25Hz, 659.25Hz, 783.99Hz)
- **特点**: 上升音调和弦

### score.wav
- **用途**: 得分时的音效
- **时长**: 0.6秒
- **频率**: A4 开始的上升音阶
- **特点**: 愉悦的上升音阶

### win.wav
- **用途**: 游戏胜利时的音效
- **时长**: 0.75秒
- **频率**: C大调主和弦分解 + 上行音阶
- **特点**: 欢快庆祝

### lose.wav
- **用途**: 游戏失败时的音效
- **时长**: 0.8秒
- **频率**: A4-G4-F4 下降音调
- **特点**: 下降音调
