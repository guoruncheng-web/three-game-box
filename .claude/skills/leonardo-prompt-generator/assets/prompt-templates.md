## 模板使用指南

### 如何使用模板

1. **选择合适的模板** - 根据需求选择最接近的模板
2. **填充变量** - 用具体内容替换 {{变量}}
3. **调整细节** - 根据需要添加或删除元素
4. **添加质量标签** - 确保包含 high quality, detailed 等
5. **设置否定提示词** - 添加要避免的元素

### 变量填充技巧

- **{{主题}}**: 具体明确，如 "cute cartoon apple" 而不是 "fruit"
- **{{风格}}**: 指定艺术风格，如 "3D render", "flat design", "pixel art"
- **{{配色}}**: 使用具体颜色名或hex值，如 "pastel pink #ffb3d9"
- **{{氛围}}**: 描述情感和感觉，如 "cheerful", "mysterious", "energetic"
- **{{光照}}**: 说明光源，如 "soft ambient light", "dramatic side lighting"

### 组合使用

可以组合多个模板元素：

```
{{游戏图标模板的主体}} + {{特殊效果}} + {{概念艺术的渲染风格}}
```

示例：
```
Cute cartoon grape cluster, 3D glossy style (from 游戏图标) +
with magical sparkle particles around it (from 特殊效果) +
rendered in Pixar animation style (from 概念艺术风格)
```
